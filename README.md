# matlab-linearity

基于 R² 线性拟合扫描的 MATLAB 信号线性区自动检测工具。

## 用途

读取一个两列 CSV 文件（时间, 信号值），自动找出信号中满足线性关系的最长连续段，并输出起止边界、R² 值和拟合直线参数。

适用于：
- ADC 斜坡信号线性度分析
- 传感器输出线性段截取
- 任意时域信号的线性区自动定位

## 快速开始

1. 将数据文件放入本目录（格式：第1行为表头，第2行起为数据，两列依次为时间和信号值）
2. 修改 `linearity_detector.m` 第5行 `fileV = 'your_data.csv'` 为实际文件名
3. 在 MATLAB 中运行 `linearity_detector`

## 可调参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `R2Thresh` | 0.9999 | R² 阈值，越高对线性要求越严格（1 = 完美直线） |
| `minLinearLen` | 10 | 最小线性区长度（点数），过短不视为线性区 |
| `smoothWin` | 5 | 差值/斜率平滑窗口（奇数），仅用于展示 |
| `rowStart` | 2 | CSV 数据起始行（跳过表头） |

## 输出

### 控制台输出
- 线性区起止索引、时间、电压值
- 拟合直线参数 `v = a*t + b`
- R² 值、斜率均值与标准差、波动百分比

### 图形输出
- **Figure 1**：全波形 + 线性区高亮 + 拟合直线标注
- **Figure 2**：差值(dv)与斜率(dv/dt) 四象限分析图
- **Figure 3**：R² 诊断图（滑动窗口局部 R² + 从起点逐点展开的累积 R²）

## 算法原理

对所有连续子区间做线性最小二乘拟合，计算 R²（决定系数），找到满足 `R² ≥ R2Thresh` 的最长连续段。该段即为信号的线性区。

当时间步长不均匀时，使用斜率 `dv/dt` 而非原始差值 `dv` 作为辅助展示。

## 依赖

- MATLAB R2019b 或更高版本（使用了 `movmedian`、`xline` 函数）
